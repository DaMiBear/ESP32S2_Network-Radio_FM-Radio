惯性传感器 (IMU)
=======================
:link_to_translation:`en:[English]`

惯性传感器包含陀螺仪传感器、加速度传感器和兼具多种功能的传感器等。主要用于测量物体的加速度、角速度等，进而解算出物体的运动姿态。
